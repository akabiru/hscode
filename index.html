hscode

hscode is a simple HTTP status code lookup tool. Lookup that status code at the comfort of your own terminal.ðŸ˜Ž

Installation

Install it yourself as:

$ gem install hscode

Or Add this line to your application's Gemfile:

gem 'hscode'

And then execute:

$ bundle

Usage

Run a quick lookup

$ hscode -c 200

For a more comprehensive description, run verbosely.

$ hscode -c 200 -v

List all status codes

$ hscode -l

In case you forget something; run help.

$ hscode -h
